# pdp_scraper 
This is a Node.js web scraper. It scrapes data from product detail pages on e-commerce sites. It handles pagination and saves all information in JSON format. Lastly, it converts this JSON information into a csv file. 

### Legal & Ethical Considerations + Best Practices
  - Do not scrape copyrighted and sensitive data
  - Limit request rate
  - Do not login
  - Follow terms of service, etc.
    
The above list does not include all important points.

### Prerequisites
Node.js installed

### Install Dependencies
npm install

### To run 
npm start
